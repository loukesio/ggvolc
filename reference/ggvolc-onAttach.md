# Print Welcome Message and ASCII Art upon Loading \`ggvolc\`

This function is executed when the \`ggvolc\` package is attached to the
R session. It prints a welcome message and an ASCII representation
related to the package.

## See also

[`library`](https://rdrr.io/r/base/library.html)

## Examples

``` r
# The function is automatically called when you use:
# library(ggvolc)
```
